#Evaluation Bash

Vous trouverez dans ce dossier :

	- Un fichier GlouMiam.txt contenant le menu personnalisable
	- Un script plat.sh permettant d'accéder au GlouMiam.txt et d'en changer le contenu de façon dynamique
	- Un script Menu.sh permettant à l'utilisateur de choisir des plats et leur nombre dans le menu pour avoir en sortie un prix HT et TTC

Amusez-vous bien mais pas trop quand même !
